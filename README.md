# Currículo (GitHub Pages) — organizado

## Arquivos (raiz)
- index.html
- styles.css
- cv.pdf

## Assets
- assets/img/profile/  -> fotos de perfil (rotaciona a cada refresh)
  - p-01.jpg
  - p-02.jpg
  - p-03.jpg
- assets/img/qr.png     -> QR do LinkedIn
- assets/svg/circles/   -> círculos em SVG (caneta)
  - circulo.svg
  - circulo1.svg
  - circulo2.svg

## Como atualizar fotos
1) Substitua/adicione novos arquivos em `assets/img/profile/` no padrão `p-XX.jpg`.
2) Atualize a lista `profileImages` no final do `index.html`.

## GitHub Pages
Settings -> Pages -> Source: main / (root)

Se não refletir:
- Ctrl + F5
- ou renomeie o arquivo (cache)
